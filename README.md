# FINMA INVEST

A web-based investment and portfolio management system with wallet integration and payment gateway.

## Features

- User authentication (JWT)
- Wallet funding (Paystack integration)
- Investment purchase using wallet balance
- Portfolio tracking
- Transaction history
- Market news integration
- Dark mode UI
- Progressive Web App (PWA)

## Tech Stack

- Frontend: React
- Backend: Node.js (Express)
- Database: MySQL
- Payment: Paystack API

## Key Functionality

- Secure payment verification
- Transaction-safe investment logic using database transactions
- Real-time-like data updates via REST APIs
